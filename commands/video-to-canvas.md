# Video to Canvas

将视频转换为 Obsidian Canvas 可视化笔记（三阶段混合管道）。

**用法**: `/video-to-canvas <视频路径> [选项]`

请先读取完整的 Skill 定义文件，然后按照指示执行：

```
Read ~/.claude/skills/video-to-canvas/SKILL.md
```

读取 SKILL.md 后，按照其中定义的完整工作流执行任务。用户提供的参数: $ARGUMENTS
